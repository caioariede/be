Toy language :)


## Mini Tutorial

### Variables

#### Format

    <expr> <id>

#### Examples

    1n.
    1 n.
    (1 + 1) n.


### Calling functions

#### Format

    <expr> id

#### Examples

    1 print
    1, 1, add, print


### Functions

#### Format

    <list> <block> <id>

#### Examples

    (n,) [ n print ] echo.
    (a,b) [ a + b ] add.
